# Don't Learn Buffett

> 先回到历史现场，再判断自己能不能做出同样的决定。

**别学巴菲特。**

不是因为 Warren Buffett 不伟大。恰恰相反，只有认真研究 Buffett，才会发现一个刺耳的事实：

**普通人最容易学到的 Buffett，往往是最危险的 Buffett。**

中文互联网上有太多“学习巴菲特”的内容：长期持有、买好公司、越跌越买、忽略波动、做时间的朋友。

问题是，Buffett 不是一个普通股票账户。他有保险浮存金、控股公司结构、永久资本、税务递延、危机中的信用、几十年积累的声誉、美国资本市场的时代红利，还有 Charlie Munger。

如果你只学他的动作，不学他的条件和判断方式，你学到的不是 Buffett，而是一个危险的幻觉。

这个项目不是一份研究报表，而是一本活的书。

它用 1957-2024 年 Buffett / Berkshire 的收益曲线、股东信、年报、市场数据和关键事件，带你回到一个个历史现场：

- 当时发生了什么？
- 如果你是 Buffett，你会怎么做？
- 你会学到表面的术，还是背后的道？
- 你有没有 Buffett 当时拥有的资金、结构、时代和心理条件？

最后再拆解一件事：

**Buffett 的成功里，哪些是能力，哪些是结构，哪些是时代，哪些是命运。**

美国、美元和国际局势不是这个项目的总论点，而是其中一组必须拆开的条件。

Buffett 主要投资美国企业，背后有持续扩张的美国消费力、以美元计价的全球利润、长期通胀下资产所有者的优势、美元作为世界储备货币带来的融资和安全资产地位，以及美国资本市场制度本身。

这些不是为了证明“Buffett 只是在买美国”，而是为了证明更大的主旨：如果不拆开能力、资本结构、时代、制度、美元和国际局势，“长期持有”“买好公司”“别人恐惧我贪婪”都会变成被抽空背景的口号。

## 项目要证明什么

这个项目不是反 Buffett，也不是反价值投资。

它反对的是把 Buffett 的结果包装成普通人可复制教程。

每个关键事件都会被写成一章。每章先把读者带回现场，再让读者自己做判断，最后才给出 Buffett 的决断和条件链。

每章都会拆成这些层次：

- 回到现场：当时的市场、公司、资金和国际背景。
- 轮到你做决定：如果你是 Buffett，你会不会做同样选择。
- 表面故事：大众通常怎么讲这个 Buffett 故事。
- 被隐藏的条件：这件事背后普通人没有的资金、结构、身份或时代条件。
- 术的陷阱：照搬这个故事最容易犯什么错。
- 道的部分：去掉神话以后，仍然值得学习的判断方式。
- 判决：一句可以被争论、也必须被证据支撑的结论。

还会增加一层宏观底稿：

- 宏观时代：这件事发生在怎样的美国和世界周期里。
- 美元背景：美元信用、通胀、利率和流动性如何影响资产。
- 美国红利：美国消费、制度、企业利润和资本市场给了什么顺风。
- 国际冲击：战争、石油、冷战、全球化、金融危机、疫情等外部事件如何改变投资者的位置。
